Customizing Bootstrap with _variables.scss

Create scss/custom.scss with this content: 
// Override Bootstrap variables BEFORE importing
$primary:       #e91e63;   // Change primary color to pink
$border-radius: 1rem;       // Rounder corners everywhere

// Then import Bootstrap
@import "../node_modules/bootstrap/scss/bootstrap";

Compile: npx sass scss/custom.scss css/custom.css --watch

After compiling, link css/custom.css instead of Bootstrap's CDN to see custom colors and rounded corners.
