Bootstrap Directory Structure

Folder/FilePurpose
css/bootstrap.min.cssMinified Bootstrap styles – link in <head>
css/bootstrap.cssFull (unminified) Bootstrap CSS for debugging
js/bootstrap.bundle.min.jsMinified JS + Popper.js bundled – link before </body>
js/bootstrap.min.jsMinified JS only (requires separate Popper.js)
icons/Bootstrap Icons SVG files (if downloaded separately)
Note: Always use bootstrap.bundle.min.js to include Popper.js automatically for tooltips and dropdowns.
