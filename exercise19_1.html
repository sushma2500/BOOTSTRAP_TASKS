Setting up Bootstrap 5 with Sass

Steps to set up Bootstrap 5 with Sass:
  1. Run npm init -y
  2. Run npm install bootstrap sass
  3. Create scss/custom.scss
  4. Add @import "../node_modules/bootstrap/scss/bootstrap"; to your .scss file
  5. Compile with npx sass scss/custom.scss css/custom.css
  6. Link css/custom.css in your HTML

See exercise19_2 for variable customization.
